# Sources

This folder defines the trusted information sources used by the
Agro Strategic Report system.

## Institutional Sources
- Spanish Ministry of Agriculture (MAPA)
- European Commission (DG AGRI)
- EFSA
- FAO
- OECD

## Technical & Scientific Sources
- Peer-reviewed journals
- Official technical bulletins
- Research institutes
- Universities

## Market & Operational Sources
- Input suppliers (fertilizers, seeds, crop protection)
- Market observatories
- Cost and price monitoring systems

## Media Filtering Rule
General media is only used if:
- It references official data
- It provides primary documentation
- It has direct operational impact

## Update Frequency
Sources are reviewed continuously with priority given to:
- Current year data
- Regulatory updates
- Campaign-relevant information
